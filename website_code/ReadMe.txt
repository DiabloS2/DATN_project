B1: Tải phần mềm XAMPP, mở xampp-control -> bật Apache + MySQL

B2: Tải folder rfid_hotel về máy tính. copy toàn bộ folder vừa tải
    vào C:\xampp\htdocs

B3: Trong folder rfid_hotel vừa tải có 1 file cơ sở dữ liệu được 
    chúng tôi tạo sẵn tên là rfid_hotel.sql.

B4: Truy cập đường dẫn: http://localhost/phpmyadmin
    sau đó tạo 1 database mới với tên là rfid_hotel
    sau đó import file "rfid_hotel.sql" vào database vừa tạo
    và ta đã có 1 database phục vụ cho việc sử dụng website

B5: Trên trình duyệt web
    Truy cập: http://localhost/rfid_hotel/
    Ta đã vào được website
